# 🍕 Online Food Ordering System

A full-stack web application for online food ordering built using Java Servlets, JSP, MySQL, and Bootstrap 5.
## 🎬 Demo Video
👉 [Click here to watch the demo](https://drive.google.com/file/d/1jbvl8h1O_h4TdbZBwRF-j88rfc7_Mk_a/view?usp=drivesdk)

## 🚀 Features

### 👤 User Panel
- User Registration and Login with Session Management
- Browse Food Menu by Category (Pizza, Burger, Pasta, Drinks)
- Add Items to Cart with Quantity Selection
- Place Orders
- Track Order Status in Real Time
- Logout

### 👨‍💼 Admin Panel
- Secure Admin Login
- View All Customer Orders
- Update Order Status
- Monitor All Transactions

## 🛠️ Tech Stack
- Java Servlets - Backend Logic
- JSP (JavaServer Pages) - Frontend Views
- MySQL - Database
- JDBC - Database Connectivity
- Bootstrap 5 - Responsive UI Design
- Apache Tomcat - Web Server
- MVC Architecture - Design Pattern

## 🔑 Admin Login
- Default admin Credentials are configured in the database setup script

## 📌 How It Works
1. User registers and logs in
2. Browses menu and adds items to cart
3. Places order
4. Admin logs in and updates order status
5. User can track order status in real time
